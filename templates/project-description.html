About the Project:

Our Real-Time Age and Gender Detection Web Application is a cutting-edge project built to showcase the capabilities of machine learning in real-time analysis. Utilizing Convolutional Neural Networks (CNNs) trained on the UTKface dataset, our application accurately predicts the age and gender of individuals in live video streams.

Features:

  • Real-Time Detection: Our application processes live video streams to provide instantaneous age and gender predictions.
  • High Accuracy: The CNN model trained on the UTKface dataset ensures accurate age and gender classification.
  • User-Friendly Interface: With a simple and intuitive design, users can easily interact with the application and view the results.

Technologies Used:

  • Flask: The web application is built using Flask, a lightweight and efficient web framework in Python.
  • OpenCV: OpenCV is utilized for video processing, allowing real-time analysis of video streams.
  • Convolutional Neural Networks (CNNs): The core of our application, CNNs are employed for age and gender prediction tasks.
  • HTML/CSS: The frontend of the application is designed using HTML for structure and CSS for styling.
